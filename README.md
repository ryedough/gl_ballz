# GL Ballz
I made this as my first C program and to practice physics  
so naturally i tried to make this app with very few library  

to build and run, just call `make` in bash  

dependency used :
- [GLFW](www.glfw.org)
- [GLAD](https://glad.dav1d.de/)
- [CGLM](https://github.com/recp/cglm)
- [my homebrew dynamic array](https://github.com/ryedough/c_dynamic_array)
- [my homebrew obj loader](https://github.com/ryedough/c_obj_loader)
